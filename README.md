TravelFlow ✈️

TravelFlow 是一款極簡、無印良品風（Muji Style）的單頁式（Single-Page）旅遊規劃 Web App。專為手機瀏覽體驗設計，提供從行程規劃、記帳、戰利品紀錄到行李清單的一站式整合服務。

🌟 核心功能特色

1. 智慧行程規劃 (Smart Itinerary)

自動日期推算：設定出發日期後，系統會自動為 Day 1、Day 2 標註對應的實際日期（例：3/1）。

多樣化活動標籤：內建飛機、交通、美食、活動、購物、住宿等專屬 Icon。

雙時區與動態追蹤：標題列即時顯示「台灣時間」與「當地時間」。系統會根據當地時間自動對比行程表，正在進行的行程會發出藍色呼吸燈光暈，已結束的行程則會自動變灰半透明。

備案系統 (Plan B)：可為每個主行程設定專屬的備案計畫。

重點標示：可將重要行程「⭐ 標示為重點」，卡片會以醒目顏色呈現。

2. 多幣種記帳與戰利品牆 (Shopping & Expense)

即時匯率換算：支援 TWD、JPY、USD、KRW、EUR、HKD。系統自動串接開源 API（Open Exchange Rates）抓取最新匯率，將外幣自動換算為新台幣。

總預算圓餅圖：自動統整行程預估花費與實際購物支出，並繪製成視覺化的預算圓餅圖。

戰利品相簿模式：購物清單支援上傳照片（內建自動壓縮機制，防止容量爆滿），並可一鍵切換為「IG 網格相簿模式」或「列表模式」。

3. 行李防呆清單 (Packing Checklist)

預設必備清單：建立新旅行時，自動載入護照、轉接頭等基本行李清單。

全域跑馬燈提醒：只要有尚未打勾的行李，系統最上方就會出現紅色跑馬燈跨頁提醒；全數準備完畢後會顯示綠色安全提示。

4. 一鍵長圖匯出 (Export as Image)

內建截圖引擎，點擊「匯出」按鈕，系統會在背景將所有天數的行程、注意事項與購物清單拼接成一張高畫質的長條圖（JPG），方便直接傳送至 LINE 或保存至手機相簿。

🛠️ 技術架構

TravelFlow 採用最純粹的網頁技術，無須安裝、無須後端資料庫，即開即用。

核心語言：HTML5, CSS3, Vanilla JavaScript (ES6+)

樣式框架：Tailwind CSS (透過 CDN 引入，快速構建響應式排版)

圖示庫：Lucide Icons (輕量級、高質感的 SVG 圖示)

圖片匯出：html2canvas (負責將 DOM 元素轉譯為圖片)

資料儲存：localStorage (資料完全儲存於使用者設備的瀏覽器中，保障隱私)

匯率 API：Open Exchange Rates (er-api)

📱 使用指南

啟動方式

這是一個純前端的單一檔案應用程式。

將原始碼儲存為 index.html。

直接使用手機或電腦的瀏覽器（如 Chrome, Safari）開啟該檔案即可使用。

（推薦）在手機瀏覽器中選擇「加入主畫面」，即可獲得如原生 APP 般的全螢幕沉浸體驗。

操作小提示 (Gestures & UI)

滑動刪除：在行程、購物或行李清單中，對著卡片向右滑動（Swipe Right）即可呼叫出紅色的刪除按鈕。

左側導航欄：手機版面下，左側設有固定的功能切換列，確保在長列表滾動時，隨時可以切換「行程 / 購物 / 行李」或執行匯出。

單一浮動按鈕 (FAB)：右下角的 + 按鈕是智慧型的。當你在行程頁面時，它會新增行程；在購物頁面時，它會新增戰利品。
